i3wm 是一个窗口管理器，平铺高效，很适合 archlinux

桌面 i3wm

安装窗口管理器

$ sudo pacman -S i3-gaps xorg-xinit

配置文件位置在 ~/.config/i3/config

官方文档 i3wm 文档 i3-gaps 文档

我的配置 https://github.com/Twor/dotfile/blob/master/i3/config

如果使用我的配置你还需要

$ sudo pacman -S feh ## 壁纸 自行更改 ~/.config/i3/config 壁纸路径
$ yay -S i3-scrot  ## 截图

效果图

启动

编辑  ~/.xinitrc，在其末尾添加这一行：

exec i3

如果想记录 i3 程序所有的输出，这一行要写成：

exec i3 -V >> ~/i3log-$(date +'%F-%k-%M-%S') 2>&1

状态栏 Polybar

安装 polybar

$ sudo pacman -S polybar

配置文件位置在 ~/.config/polybar/config

官方文档 polybar 文档

我的配置 https://github.com/Twor/dotfile/tree/master/polybar

启用：

添加一个启动脚本：$HOME/.config/polybar/launch.sh

#!/usr/bin/env bash

# Terminate already running bar instances
killall -q polybar
# If all your bars have ipc enabled, you can also use
# polybar-msg cmd quit

# Launch bar1 and bar2
echo "---" | tee -a /tmp/polybar1.log /tmp/polybar2.log
polybar bar1 >>/tmp/polybar1.log 2>&1 &
polybar bar2 >>/tmp/polybar2.log 2>&1 &

echo "Bars launched..."

请在在 i3 的配置文件中添加以下几行：

exec_always --no-startup-id $ HOME /.config/polybar/launch.s

并删除这些

bar {
    i3bar_command i3bar
}

第一行执行启动脚本；第二个禁用 i3 的默认栏

窗口模糊 picom(compton)

因为 compton 难以维护，所以分支了这个项目。而需要模糊效果就安装大神修改的版本，如果你需要半透明、透明则可以安装官方版。

安装 picom

$ yay -S picom-tryone-git

配置文件位置在 ~/.config/picom/config

官方文档 picom 文档

我的配置 https://github.com/Twor/dotfile/blob/master/picom/picom.conf

主题和字体

主题

这个没啥好说的，安装，使用就完事！

我使用使用arc-gtk-theme界面主题和numix-icon-theme图标主题，配置工具我是使用的lxappearance。安装

$ sudo pacman -S arc-gtk-theme numix-icon-theme-git lxappearance
字体

如果你使用的我配置，那么请安装下面的字体

$ sudo pacman -S nerd-fonts-source-code-pro noto-fonts noto-fonts-cjk noto-fonts-emoji noto-fonts-extra
$ yay -S ttf-ms-fonts ttf-cascadia-code

解压

sudo pacman -S p7zip file-roller unrar
